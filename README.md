# 莲花派奇门遁甲 · 在线排盘

时家转盘奇门 · 拆补法定局 · 莲花十二宫体系。纯静态网页，零依赖，浏览器打开即用。

## 功能

- **输入**：公历年月日时分
- **真太阳时校正**：经度时差 + 均时差（Equation of Time），内置常用城市经度，支持自定义经度
- **四柱**：自实现干支历（五鼠遁时干、五虎遁月干、立春换年、节气换月、23 点归次日）
- **定局**：拆补法（二十四节气精确时刻 + 符头定上中下三元）
- **转盘**：九星随时干、八门随时宫、八神小符随大符
- **标注**：入墓、击刑、门迫、刑+墓、空亡、马星

## 文件

- `index.html` — 页面与交互
- `qimen-core.js` — 排盘核心引擎（节气天文算法、四柱、定局、转盘、标注）

## 部署

纯静态站，无需构建。可直接用 Cloudflare Pages / GitHub Pages 等托管，构建命令留空，输出目录为根目录。
